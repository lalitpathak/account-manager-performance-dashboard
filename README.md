**📊 Regional Account Manager Insights 2023**
An executive-style Tableau dashboard designed to analyze account manager performance, customer portfolios, sales pipelines, acquisitions, renewals, and revenue trends.The dashboard combines business intelligence, KPI tracking, and modern dashboard design principles to deliver an intuitive analytical experience for performance monitoring and decision-making.

**Project Overview**
This project focuses on providing actionable insights into account management operations by analyzing:

    portfolio performance
    customer acquisition
    sales pipeline health
    revenue trends
    conversion efficiency
    renewal opportunities

The dashboard was designed with an emphasis on:

    executive readability
    interactive exploration
    modern UI-inspired analytics
    clean visual hierarchy

**📂 Dashboard Components**

| Section | Purpose |
|---|---|
| Performance Rank | Displays comparative ranking of account managers |
| Portfolio Overview | Summarizes customer portfolio and pipeline health |
| Pipeline Analysis | Tracks deal stages, opportunity distribution, and pipeline value |
| Conversion Rate | Monitors year-over-year opportunity conversion performance |
| Revenue Trends | Analyzes annual revenue performance over time |
| Top Accounts | Highlights highest-value customer accounts |
| Renewals | Tracks upcoming customer renewal opportunities |
| Acquisitions & Losses | Monitors new business acquisitions and lost opportunities |
