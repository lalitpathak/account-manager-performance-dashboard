## Objective

The objective of this dashboard was to build an executive-level business intelligence solution capable of monitoring regional account manager performance, pipeline health, customer portfolios, and revenue trends through an interactive analytical interface.

The design focused on:
- executive readability
- visual hierarchy
- modern UI aesthetics
- performance storytelling

---

# Design Philosophy

The dashboard was designed using a KPI-first approach where users can immediately evaluate performance metrics before exploring detailed business insights.

Special emphasis was placed on:
- reducing dashboard clutter
- maintaining analytical clarity
- creating a professional enterprise-style layout
- balancing aesthetics with usability

---

# Layout Strategy

The dashboard layout follows a structured analytical flow:

1. Account Manager Selection
2. Performance KPIs
3. Pipeline & Portfolio Insights
4. Revenue & Conversion Trends
5. Customer Opportunity Analysis

This improves navigation and supports intuitive exploration.

---

# Visual Design Choices

## Color Palette

A soft enterprise-style color scheme was selected to:
- maintain professionalism
- reduce visual fatigue
- improve readability
- create visual consistency

## Dashboard Components

Rounded KPI containers and bordered analytical sections were used to create:
- clear separation between insights
- modern dashboard styling
- organized visual grouping

---

# Tableau Techniques Used

- Interactive dashboard navigation
